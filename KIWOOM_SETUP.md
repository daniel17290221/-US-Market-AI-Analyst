# 키움증권 REST API 연결 설정

현재 연동 범위는 조회 전용 연결 확인 단계입니다. 주문 기능은 포함하지 않습니다.

## 1. 키움증권에서 준비

1. 키움 REST API 사용을 신청합니다.
2. 모의투자 또는 운영용 App Key와 Secret Key를 발급받습니다.
3. 키움 개발자센터에서 허용 IP 등 현재 적용되는 보안 조건을 확인합니다.

## 2. 서버 환경변수

로컬 `.env` 또는 Vercel 프로젝트 환경변수에 아래 값을 등록합니다.

```text
KIWOOM_APP_KEY=발급받은_App_Key
KIWOOM_SECRET_KEY=발급받은_Secret_Key
KIWOOM_API_MODE=mock
KIWOOM_SYNC_SECRET=충분히_긴_임의의_관리용_비밀값
```

`KIWOOM_API_MODE`는 연결 검증이 끝날 때까지 `mock`을 사용하고, 실계좌 연결 시에만
`production`으로 변경합니다. 어떤 비밀값도 Git에 커밋하지 않습니다.

## 3. 연결 상태 확인

- `GET /api/kiwoom/status`: 키 설정 여부와 모드만 반환하며 키움 서버를 호출하지 않습니다.
- `POST /api/kiwoom/verify`: 실제 토큰 발급으로 연결을 검증합니다.
- `POST /api/kiwoom/snapshot`: 마스킹된 계좌번호, 예수금, 국내 보유종목을 조회합니다.

검증 요청에는 다음 헤더가 필요합니다.

```text
X-Kiwoom-Sync-Key: KIWOOM_SYNC_SECRET에 설정한 값
```

`verify` 응답에는 접근 토큰, 앱키, 시크릿 또는 계좌정보가 포함되지 않습니다.

`snapshot` 응답에도 전체 계좌번호는 포함되지 않으며 마지막 네 자리만 표시됩니다.
현재 단계는 국내주식 조회 전용이고 주문 API는 연결하지 않습니다. 웹 화면에 입력한
연동 PIN은 요청 직후 입력란에서 삭제되며 로컬 저장소나 쿠키에 저장하지 않습니다.

## 거래내역과 수익률

`snapshot` 요청은 기본적으로 최근 12개월의 키움 위탁종합거래내역(`kt00015`)을
조회하여 FIFO 방식으로 실현손익, 미실현손익, 세후 배당과 총수익률을 계산합니다.
화면에서 12개월, 36개월, 60개월을 선택할 수 있습니다.

키움 조회기간보다 오래된 매수분은 화면의 **과거 매수내역 직접 보완**에서 입력할 수
있습니다. 이 내역은 해당 브라우저의 `localStorage`에만 저장되며 서버에는 계좌 조회
요청을 계산하는 동안에만 전달됩니다. 거래내역이 부족한 잔여 수량은 키움 평균매입가로
추정되며 화면에 `일부 내역은 증권사 평균단가로 추정`이라고 표시됩니다.
